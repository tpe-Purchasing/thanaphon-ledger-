บันทึกรายการ

บันทึกเอกสารรับเข้าต่อเนื่อง ข้อมูลซิงค์ทุกเครื่องแบบเรียลไทม์

แนะนำให้ใช้ลิงก์จาก File → Share → Publish to web (เลือกเฉพาะแท็บ "บันทึกรายการ" รูปแบบ CSV) จะดึงข้อมูลได้ครบและเสถียรที่สุด ระบบจะข้ามรายการที่มีบริษัท+เลขที่+วันที่ตรงกับที่มีอยู่แล้ว กันข้อมูลซ้ำเวลากดดึงหลายครั้ง กดซ้ำได้เรื่อยๆ เมื่อมีการอัปเดตในชีท

รองรับเครื่องหมาย ✓ หรือ TRUE ในคอลัมน์ใบกำกับ/สำเนา ระบบจะรันเลขลำดับใหม่ให้อัตโนมัติต่อจากรายการล่าสุด
